Amnesiac Engine Documentation

The Amnesiac Engine is a 2D, sprite-based game engine.
This documentation primarily targets version 0.1.5 alpha and up of the engine.


Running the engine

To run the engine, (assuming it's installed correctly) all you need is to create a launch script as follows.
game_path is the path to the game data folder, relative to the launch script. It is reccommended to have the launch script as well as the game data in the same directory.

import os
from amnesiac.engine import RunEngine
RunEngine(game_path, os.path.dirname(__file__))
